バージョン番号は、0.1 辺りから 0.01 づつシリアルに上げて
いっただけなので、特に意味はないといえばないですが、何となく
1.0 になった瞬間に好きにいじれないような気がして、
簡単に 1.0 だと言いたくないという気持ちもあります。

# 僕の場合、大抵 0.3 やそこらが一番旬だったりします。:-)

elisp の mhc は皆さんの関心も高くて、沢山助けていただいて、
結構いい感じになって来たと思います。今の所の大物は、

 ドキュメントの整理。
 elisp でいうところの major-mode になる。
 alarm 関係。
 各種補完。
 iCalendar の MIME Object とのインターフェース。

辺りですね。あと、英語メールからも guess できればいいと思ってい
ますが、英語のスケジュールメールなんて飛んで来ないもので。。

MHC 全体としては、Ruby のライブラリを使って色々面白いことをやり
たいなと思っています。

shell スクリプトとからも呼べるような小物集とか。
TV ガイドのデータで、少し遊んだ形跡が入っていますが、そういうのです。
note と desktop との間の sync なんかもできるといいですね。

Web との連携やグループウェアサーバのような、ややお仕事でも使え
そうな物も考えています。こっちは、側に座っている人が
頑張ってくれているので、どうなるか楽しみにしているところです。
