年明けに Palm m100 を買いました。
で、mhcと連携させようと何度かやってるのですが、
どうにもうまくいきません。
時間待ちの間にやってみましたが、やっぱりダメ。。。
さて、どうしましょう。

1. rbcw145.zip をc:\rbcw145 に展開
;; なぜ1.4.5かって??
;; だってドキュメントがないんだもの。
2. autoexec.batを編集
   set PATH=C:\rbcw145;%PATH%;
   set RUBYLIB=//c/rbcw145/lib
3. 再起動
4. ruby-ext/lib 以下のファイルを、文字コードをsjis-dosに変更して
   c:\rbcw145\lib にコピー。
   mhc-date.rb
   mhc-gtk.rb
   mhc-kconv.rb
   mhc-palm.rb
   mhc-schedule.rb
5. （多分以前作った）mhc2palm、palm2mhc を、
   ファイルの先頭を
   #!/rbcw145/ruby -Ke
   に書き換えて、c:\usr\local\binにコピー。
6. mhc_pilib.dll（size 65203） を
   c:\rbcw145\lib にコピー。
7. mhc2palm -a -d COM4 すると以下のエラー
/USR/LOCAL/BIN/mhc2palm:18:in `require': //c/rbcw145/lib/mhc-palm.rb:648: parse error (SyntaxError)
//c/rbcw145/lib/mhc-palm.rb:672: return appeared outside of method
//c/rbcw145/lib/mhc-palm.rb:749: parse error
	from /USR/LOCAL/BIN/mhc2palm:18
