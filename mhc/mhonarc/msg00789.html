僕は rc.conf を書換えるのが普通なのかなと思っていたんで、
書換えるのがダサいという意識はなかったです。

# もっというと、/usr/local/pilot/lib に
# .so を置くのが好みなじゃないんです。

ところで -rpath というのは、FreeBSD (GNU ld) 以外でも OK なので
しょうか。rpath 相当のオプションって ld によって違ったり
しそうなんで。(Solaris 2.6 の ld にはこのマニュアルが見当たらない。)

create_makefile() する前に

  CONFIG["LDSHARED"] = "LD_RUN_PATH=#{found_path} " + CONFIG["LDSHARED"]

という具合にしようかと思っています。あとでコミットします。
手元にはこの版の extconf.rb があります。

pilot-link の発見に失敗したら、lib/*.rb を install する
ターゲットだけの Makefile を作るようにもしました。
