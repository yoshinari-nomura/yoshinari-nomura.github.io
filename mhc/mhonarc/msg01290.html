確認しました。見事に落ちますね。
Ruby 1.4.5 だと大丈夫みたいです。
普段 1.4 系で使っていたので気付きませんでした。
Hash の扱いが変わっているようで、

  h = Hash .new
  h['X'] = nil
  h .keys .each{|key| p h[key] }

この動作が 1.4 系と 1.6系で違うのが原因みたいです。
mhc-signal.rb の

  def signal_disconnect(sd)
    @proc_table .each_key{|sig|
      @proc_table[sig][sd] = nil
    }
  end

の部分を

  def signal_disconnect(sd)
    @proc_table .each_key{|sig|
      @proc_table[sig] .delete(sd)
    }
  end

とするとうまく動くと思います。
