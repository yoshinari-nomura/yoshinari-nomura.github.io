SL-C700 だと，emacs + mew + mhc がまともにキーボードから操作でき
るので，それで満足しています．

# 仮名漢字変換は SKK派なので，特に苦労していません．

A300 も，ビューワとして使う分には，結構いけます．Ruby と ssh 
で，mhc-sync もいけます．純粋なビューワなら，rsync もいいでしょう．

下のような .emacs を書いて，

  http://www.kick4.net/~nom/diary/exhibition/dot-emacs-for-zaurus.txt

できるだけペンを使わないでいいようにしています．
端末的には，ペンのタップで xterm-mouse-mode に対応したシーケンス
を吐くようなので，頑張ればペンから使えるような UI が書けるかもしれません．

ちなみに，

  alias e='TERM=xterm-new; set +m; emacs' 

で，Resume 後画面が崩れるのも防いて，色付きになります．
ただ，アイコンが出ないのはちょっと寂しいですね．
