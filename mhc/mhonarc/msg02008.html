多分に「好みの問題」と言われてしまいそうですが...

  [data] -> [time] -> [subject] -> [location] -> [catetory] -> [alarm]
という順序になっていますが、 [data], [time], [location] の入力は連続し
て欲しいと思っています。

とりあえず、
  [data] -> [time] -> [location] -> [subject] -> [catetory] -> [alarm]
とするようにしてみました。
([data], [time], [location] が連続していることだけが希望なので、[sub.]
 が最初でもかまいません)
