最近、H" の機種変更をして Panasonic KX-HV210 にしつつ、メール放
題(500円/月)にも加入しました。

この KX-HV210 は Outlook の予定表を "schedule.csv" という名前の
csv ファイルに落したのち、SDカード渡し or (ちょっと制限事項の多
い)メールの添付ファイルで送ると、HV210 本体の予定表に取り込むこ
とができます。

today を改造して、直接、スケジュールをメール送信できるようにした
ので、(折角だし) ML にながします。

気の効いている点
(1) Outlook => HV210 だと繰り返しの予定が一日目しか表示されない
    が、毎日表示されるようにしました。
    → 正月休みがいつ終わるかわかります。

(2) Outlook => HV210 だと、アラームのなる時間が開始時刻固定で、
    オフセットが無視されるが、開始時刻をアラーム分前倒しにするこ
    とにより対処しました。そのときは Location 部分に実際の開始時
    刻が表示されます。
    
(3) Script 中の DummyMailAddr を適切に設定しておけば、例えば
    % today-hv210 --date=today+30 > ~/Mail/tmp/100
    とかした後、Reedit 使って送信すれば Windows でも簡単。

野暮な点
(1) アラーム分前倒ししたときに前の日になっちゃったら、夜中の
    "00:00" にアラームがなります。

(2) today の持っていた --format オプションは無くして HV210 専用
    にしてしまいました。

(3) Ruby って良くわからないので、良くわかっていないところが多々
    あります。
    
というわけで、使ってくれる人がいるかどうかは不明ですし、類似の機
種で他に有効なものがあるかもわかりませんけど、楽しんでくださいませ。
