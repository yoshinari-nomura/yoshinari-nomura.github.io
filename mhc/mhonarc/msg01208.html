00readme.jis のインストール手順が正しくないようです．

Ruby 1.6.2 を使っていますが，ruby 要のライブラリが正しくインストールされ
ません．

  ruby configure.rb
  ruby make.rb
  ruby make.rb install

とありますが，この手順だと，正しい mhc-kconv.rb がインストールできないよ
うです．

一旦 ruby configure.rb で正しい mhc-kconv.rb が生成されるのですが，その
後 ruby make.rb で正しくない mhc-kconv.rb が上書きされます．

ruby make.rb を飛ばせば，少なくとも mhc-kconv.rb は正しいものがインストー
ルされます．ただ，configure.rb, make.rb がなにを実行しているのかよく分か
らないので，ドキュメントをどう変更するべきなのかはよく分かりません．

また，ruby-ext/Makefile も install ターゲットが DLLIB に依存していますが，
configure.rb のオプションによっては，DLLIB はインストールされないと思い
ます．ですから，この依存関係は良くないと思います．
