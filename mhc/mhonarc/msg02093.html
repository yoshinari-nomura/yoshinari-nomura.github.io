の扱いで問題がでてくることが分りました。

を防ぐため、japanese-hankaku()の Help (※1)を参考にして

Optional argument ASCII-ONLY non-nil means to return only ASCII character.
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
