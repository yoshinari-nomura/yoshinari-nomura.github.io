wysihtmlモードというのを作成してみました．
編集中のHTMLをmozillaでプリビューするというものです．

elisp と C のプログラムで書かれています．
C の部分でmozillaとのインタフェースをもたせ
elisp の部分でemacsでの入力検出と
編集中のファイルのhttp経由での提供を行っています．

whizzytex と比較すると，微妙にまだまだという感じがしますが，
それなりに使えるかと思います．

http://www.netfort.gr.jp/~dancer/software/wysihtml.html.ja
にウェブページ(スクリーンショットつき)が，

http://www.netfort.gr.jp/~dancer/software/downloads/list.cgi#wysihtml
http://www.netfort.gr.jp/~dancer/software/downloads/wysihtml-0.3.tar.gz
にソースがtar.gzでおいてあります．



./configure && make && make install して，
/usr/local/share/wysihtml/wysihtml.el をemacsで読み込んで，
html-mode のバッファで M-x wysihtml-mode とすると，
起動します．

すでに，mozilla が起動している必要があるようです．
また，動作には elserv が必要です．
