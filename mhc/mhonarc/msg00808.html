乃村です。cmail 関係がもうちょっとだけ進みました。

emacs/00usage.jis の 「3. .emacs に以下のように記述します。」
の部分を

  (autoload 'mhc-cmail-setup "mhc-cmail")
  (add-hook 'cmail-startup-hook 'mhc-cmail-setup)

のようにします。C-c| や C-c{n,p} SPC は効きます。
実験なので、mhc-cmail.el の中で、cmail の関数を上書きしています。
load の順番に注意して下さい。

既知の問題点:

  cmail のフォルダとは無関係に ~/Mail を掘ります。
  重なる場合は mhc 側をカスタマイズして除けて下さい。

  ~/Mail/ の mode をちゃんと指定していないので、700 にするなりして下さい。

  勘で作ったので、cmail の folder の内容も壊す可能性があります。
  バックアップを忘れずに。 

  cmail には folder-mode-hook がない。

  色々な場面で、folder を作りますか? と聞かれる。
    yes と答えてはいけません。
    (その operation は mhc からはまだ使えないということです。)

  r や f が効かない。(gnus-cmail を参考に作るのがいい?)

  C-cf が mhc によって奪われる。
    minor mode に譲って欲しいな。

  全 folder で **cmail-summary** を共有するので、
    v したときに、mhc が作った buffer-local な変数を消せない。
