mhc-mode の key binding を変えました(CVS の先端の物です)。
デフォルト設定では

旧	=> 新
C-c.	=> C-c..
C-ce	=> C-c.e
C-cn	=> C-c.n
...以下続く... 

のように、'C-c' が 'C-c.' に変わっています。また、この prefix は
mhc-prefix-key という変数で設定出来るので、従来通りが良い人は
~/.emacs で

(setq mhc-prefix-key "\C-c") 

としてください。

# mhc が load された後では変更できないので、Emacs を再立ち上げし
# てください。

なお、mhc-mode で M-x describe-mode したり、menu で mhc を選べば、
現在の key bind がわかるので参考にしてくださいませ。

## ちなみになんで変えたか興味のある方は、Emacs Lisp Info の
## 'Major Mode Conventions' や 'Minor Mode Conventions' を見てく
## ださい。mhc は minor mode です。^^;;;
