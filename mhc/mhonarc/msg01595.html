は下に追いやる option つくりましょうか？
□ <= Todo, ■ <= Todo + Done として、

○ mhc-todo-sort-method => nil 
従来通り、まったく sort しない。


○ mhc-todo-sort-method => 'deadline
今回のパッチと同等。

□ 仕事0(2日超過)
□ 仕事1(あと1日)
■ 仕事2
□ 仕事3(あと3日)
■ 仕事4
□ 仕事5(あと5日)


○ mhc-todo-sort-method => 'unfinished
