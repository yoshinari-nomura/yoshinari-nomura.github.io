TODO list を使っていますが、次のような機能を実現できないでしょうか。

・done になった TODO は TODO list には表示しない。
・通常の予定には終わった done になった TODO も表示したい。
  (TODO も予定の一部なので、後から確認したいので)

具体的には、
通常の予定部分を表示するときの category 指定(mhc-default-category)以外に
TODO list を表示するときの category を別に指定できれば、と思っています。


ちなみに、現在の私の使用方法はこんな感じです。
(1) done になった TODO は Category: Todo Done とする。
(2) mhc-default-category = "!done" と設定し、TODO list に done になった
    TODO は表示されないようにする。

全ての予定を確認したいときは、default-category を変更して rescan しています。

よろしくお願いします。
