> cygwin 側の permission 設定はどうでしょう?.

cygwin の install から始めたんではないかと思い, 上記文章を書きました. 
当方は, mhc では無いですが cygwin の install をしていて permission ま
わりではまりました.

cygwin ではいつ頃からかは不明ですが環境変数 CYGWIN が設定されていない
場合, ntsec を仮定する様になった模様です. ですから, NT の権限 (セキュ
リティポリシーと言うんでしょうか.) が正しく設定されていないと, パーミッ
ションが変(?) になります. 

めんどくさがりの佐々木は, CYGWIN=nonntsec にして, cygwin の再 install から
始めました.
