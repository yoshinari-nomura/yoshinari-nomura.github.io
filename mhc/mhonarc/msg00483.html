ノートパソコンとの間のスケジュールの同期についての話題です。これは私も
非常に欲しい機能ですので、大急ぎででっち上げてみました。

要点は簡単で、スケジュールを管理する全てのファイルを CVS の管理下に置
いておきます。

    cd ~/Mail/schedule
    cvs import -m "Initial Schedule" schedule username start
    cd ..
    rm -rf schedule
    cvs co schedule

これだけ準備をしておいて、添付のパッチを適用した MHC でスケジュール管
理を行うと、ファイルの管理を CVS を通して行うようになります。

一旦、CVS で管理するようにしてしまいさえすれば、母艦PCとノートの間のス
ケジュールの同期などは簡単なスクリプトで実現できると思うのですが。
