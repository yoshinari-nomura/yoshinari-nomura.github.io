今、いろんなことに手を出して、全部中途半端になってます。:-)

一部で話題になっている cmail の support を入れようと思っ
ています。少し作業をして current に叩き台を入れました。

emacs/00usage.jis の

3. .emacs に以下のように記述します。の部分を

  (autoload 'mhc-cmail-setup "mhc-cmail")
  (add-hook 'cmail-startup-hook 'mhc-cmail-setup)

のようにして実行すると、一応 summary が出るところまでは、できました。

C-c| や C-c{n,p} 等は効くと思います。
誰か確認できる方、お願いします。(って、ここにそんな人はいないか。)

問題点:

  space を押しても、登録した予定のメールがまだ読めません。:-)

  cmail のフォルダとは無関係に ~/Mail を掘ります。

  重なる場合は mhc 側をカスタマイズして除けて下さい。

  Mail/ の mode をちゃんと指定していないので、700 にするなりして下さい。


  勘で作ったので、cmail の folder の内容も壊す可能性があります。
  バックアップを忘れずに。 

  cmail には folder-mode-hook がない。

space を押したら、該当するメールが読めるようになれば、
一応使えるようになると思います。が、ここがちょっと大変そう。
