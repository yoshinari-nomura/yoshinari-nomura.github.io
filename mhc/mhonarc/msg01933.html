today の --mail オプションでメールを送ると、Date フィールドがつかない
ためでしょうか、日付が 00/00(?) 09:00 になってしまうようです（わたしの
環境は qmail+courier-imap+Wanderlust といった環境です）。
そこで、わたしは　date -R の実行結果を挿入するように修正しています。
ささいなことではありますが、ご参考まで。
もっと違う修正方法があればご指摘ください。

--- 254,260 ----
    header    = "To: #{mail_address}\n"
    header   += "From: secretary-of-#{mail_address}\n"
    header   += "Subject: Today's schedule (#{date_from .to_s})\n"
+   header   += "Date: #{`date -R`}\n"
    header   += "\n"
    header   += "#{user_name}'s schedule: \n\n"
    contents  =
