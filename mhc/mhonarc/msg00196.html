乃村です。
mhc 0.22 からかなり間が開きました。ぼちぼちやっています。

 http://www.swlab.csce.kyushu-u.ac.jp/~nom/prj/mhc/mhc-snap19991028.tar.gz

0.23 alpha という位置付けにしています。
けっこう忘れてしましたが、以下のような変更点があります。かなり変
わりました。設定方法も違います。emacs/00usage.jis をもう一度読ん
で下さい。関数の名前もそのうち色々変わるかもしれません。

mscan を使わなくなったバージョンですが、けっこう快適です。
マシンにもよると思いますが、僕の場合は scan が速くなりました。
全部 alist でキャッシュしています。

最初の 1回だけ .schedule/intersect/該当月 を全部見に行くので、ちょっ
と重いかもしれません。

入力の補完は、そのためのコードが換装したハードディスクの中に眠っ
ているのを忘れていました。週末に取り出せたらくっつけます。

xemacs だと overlay とか text-property 周りでこけるかもしれません。

あと、ちゃんと minor mode map を使うようにしてみたので、
多分、真の minor-mode になったと思います。
mhc-mode を on にしといてやると任意のバッファで  C-c. が聞きます。

* 0.23 alpha (1999-8-28)

・ gemcal の安定化
・ Palm/WorkPad との転送コマンド (palm2mhc, mhc2palm) の追加
・ mhc.el -- mscan2 を使わないバージョン。

    + スケジュールの登録 (C-cC-c) 段階で事前に
      コンフリクトの警告をするようになった。
    + カテゴリと face の関係を書く alist の追加。
    + 任意のバッファにスケジュール一覧を挿入する
       mhc-insert-schedule の追加。
    + スケジュールの登録 (C-cC-c) のフォーマットのチェック強化
    + Private なスケジュールのサブジェクト部分を [SECRET] のように
      別な文字列に変えて表示できるようになった。
    + メールからの取り込みの際、日付推測にひっかかった部分を
      ハイライト表示してみせる。
    + minibuffer 入力のチェック強化
    + minibuffer 入力のフォーマット変更により一時的に入力が不便に。

    + emacs 19.28 に対応。
