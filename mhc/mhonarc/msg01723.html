> Linux (Debian 3.0) 上で User-Agent: にあるような Emacs 環境で
> 便利に MHC を使わせていただいています。MHC のバージョンは debian の
> パッケージになっている 2002/07/10 の CVS のものです。
> 
> X-Face-1.3.6.19 と一緒に使っているのですが、xface を含んでいる
> メイルが import できません。
> C-c . l (mhc-import) を実行すると、そのまま固まってしまいます。
> *Messages* のバッファにも何も表示されません。

WL は全然わからないのですが、Emacs-21.2 + wl-2.11.0 +
x-face-e21 は問題なく import できました。

ちなみに、C-g で抜けられるなら M-x toggle-debug-on-quit してから
固めて、どこで止まっているのか調べることができます。


P.S.
そういえば、Mew で X-Face や smiley-mule 使っている人がどうやっ
て import しているのか不思議だった(ぼくは勝手にやっていた)のです
が、

mhc-mew-redisplay-import

という変数を増やして、そいつが non-nil なら X-Face の表示や
message-hook をはずして re-display するようにしてみました。
commit はあとでやっておきます。default は nil で従来動作と一緒で
す。

# quickhack.net が生き返ったので、すでに commit しました。
