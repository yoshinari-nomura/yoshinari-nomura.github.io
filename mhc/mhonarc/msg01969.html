# 新規変数を作るかどうか悩んだのですが。。。

・mhc-use-month-separato => t
いままでと同じで表示する。

・mhc-use-month-separato => nil
いままでと同じで表示しない。

・mhc-use-month-separato => 数字
separator の長さを指定しつつ表示する。week-separator と
month-separator が重なるときは week-separatorと同じ長さになり、
CW も表示します。
